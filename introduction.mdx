## Key Features

EVO Documentation stands out with its intuitive interface and robust feature set. The platform supports multiple documentation formats, real-time collaboration, and seamless integration with popular development tools. You can create structured documentation spaces, manage permissions, and track changes with comprehensive version control.

<Steps>
  <Step title="Create Your Space" icon="folder">
    Start by creating a dedicated documentation space for your project. This provides a structured environment where you can organize content by topics, versions, or teams.
  </Step>
  <Step title="Invite Team Members" icon="users">
    Collaborate effectively by inviting team members with appropriate permissions. EVO supports role-based access control, ensuring the right people have the right level of access.
  </Step>
  <Step title="Import Existing Content" icon="upload">
    Bring your existing documentation into EVO using our import tools. We support various formats including Markdown, HTML, and popular documentation platforms.
  </Step>
  <Step title="Customize and Publish" icon="settings">
    Tailor your documentation to match your brand and publish it instantly. EVO provides customizable themes, search functionality, and analytics to track engagement.
  </Step>
</Steps>
