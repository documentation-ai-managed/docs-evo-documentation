## Content Management

Organize and manage your documentation with flexible content management tools.

<CodeGroup tabs="Organize,Template,Publish">
  ```javascript
// Organize content by spaces and folders
const space = await evo.spaces.create({
  name: 'Product Documentation',
  folders: ['Getting Started', 'API Reference', 'Tutorials']
});
  ```
  ```javascript
// Use templates for consistency
const template = await evo.templates.create({
  name: 'API Endpoint',
  fields: ['method', 'endpoint', 'description', 'parameters']
});
  ```
  ```javascript
// Publish with custom domains
const publication = await evo.spaces.publish(spaceId, {
  domain: 'docs.mycompany.com',
  theme: 'corporate'
});
  ```
</CodeGroup>

## Security and Permissions

Keep your documentation secure with granular permission controls.

- **Role-based Access Control**: Assign roles like Viewer, Editor, and Admin to control access.
- **Space-level Permissions**: Set permissions per documentation space or folder.
- **Audit Logs**: Track all access and changes for compliance.
- **SSO Integration**: Connect with your identity provider for seamless authentication.

<Callout kind="info">
  EVO complies with industry standards like SOC 2 and GDPR for enterprise security requirements.
</Callout>
